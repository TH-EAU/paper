Votre premier composant

Dans ce chapitre nous allons créer notre premier composant et comprendre les concepts nécessaires pour que vous puissiez créer vos propres composants 👍 !

Phase d’analyse

Dans le cours précédent, nous avions plongé dans l’architecture de React. Il nous reste beaucoup de mystère à élucider, mais voici le premier :

import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import './index.css'
import App from './App.tsx'
 
createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <App />
  </StrictMode>,
)

Ici, au milieu de votre main.tsx, se trouve votre premier composant : App.tsx

<StrictMode/> est aussi un composant, mais si vous observez bien, il est importé de React tandis que nous avons accès à App.tsx dans nos fichiers.

🚨🚨 Et là STOP !! 🚨🚨 Nous n’allons pas explorer ce fichier pour l’instant car il est destiné à des personnes qui connaissent React.

Nous allons supprimer la ligne correspondante.

import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import './index.css'
// hop la 🪄
 
createRoot(document.getElementById('root')!).render(
  <StrictMode>
  // et hop... 🪄
  </StrictMode>,
)

Avant de vous embrouiller créeons ensemble notre premier composant.

Créer un composant = déclarer une fonction

On l’as dit précédemment, les composants Reacts sont des fonctions TS. Le nom de ces fonctions doit IMPERATIVEMENT avoir une première lettre en majuscule, pour ne pas être confondu avec des fonctions classiques.

Donc pour créer notre premier composant, vous pouvez créer un fichier dans src/ que vous appelez MyFristComponent.tsx. Vous y inscrivez :

const MyFristComponent:React.FC = () => {
 return null
}
 
export default MyFirstComponent

Ici, que du JS. A ce stade on ne fait pas de distinction excepté celle induite par TypeScript qui indique que cette fonction renvoie un React.FC, qui correspond à une fonction React.

Je peux déjà me servir de mon composant dans main.tsx !

import { StrictMode } from 'react'
import { createRoot } from 'react-dom/client'
import './index.css'
import MyFirstComponent from './MyFirstComponent.tsx'
 
createRoot(document.getElementById('root')!).render(
  <StrictMode>
	<MyFirstComponent/>
  </StrictMode>,
)

Et nous pouvons observer le résultat et … rien.

Mais c’est tout à fait normal, le résultat de notre fonction est Null ce qui indique à React de ne rien afficher. Modifions notre composant pour qu’il puisse afficher quelque chose :

const MyFristComponent:React.FC = () => {
 return (<p>Hello World</p>)
}
 
export default MyFirstComponent

En inscrivant du TSX dans le return, mon composant se met maintenant à nous afficher “Hello world”.

Le TSX étant très proche du html, vous avez déjà compris comment mettre à profil ce nouvel outil !

Conclusion

Vous venez d’apprendre à :

  • créer un composant
  • importer un composant

Il reste encore beaucoup de concepts à aborder mais vous êtes sur la bonne voie !
