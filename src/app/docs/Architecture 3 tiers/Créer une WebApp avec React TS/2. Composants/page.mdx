# Les Composants en React

*Résumé du chapitre : React propose un système de composants sous forme de fonctions JS. Celles-ci renvoient du JSX, langage déclaratif permettant d'intégrer du JS dynamique directement dans le HTML. Les composants ont un cycle de vie, qui leur permet de mettre à jour leur **state** ou **état**. Chaque changement d'état déclenche la mise à jour du composant, et donc de son rendu.
